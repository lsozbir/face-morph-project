# face-morph-project

Face morphing implementation for privacy protection

Instructions:
run main.py for a demo of face morphing to protect your privacy!

Requirements
- A webcam
- Python 3.12.7
- mediapipe 0.10.18
- opencv-python 4.10.0.84
- opencv-python-headless 4.10.0.84
- scipy 1.14.1
- deepface 0.0.93
